# Coin Stats

Live demo: https://coin-stats-seven.vercel.app/

A responsive cryptocurrency dashboard built with React and Tailwind CSS. Shows market data, coin details, price charts, search and sorting, with a clean mobile friendly UI.

## Features
- Live market overview with paging and sorting
- Detailed coin page with price chart
- Search coins by name or symbol
- Fully responsive Tailwind CSS design
- Smooth routing using React Router
- Fast deploy on Vercel

## Tech Stack
- **React**
- **Tailwind CSS**
- **React Router**
- **Fetch or Axios** for API calls
- **Vercel** deployment
